# Brian Eng — Portfolio

A static single-page portfolio site built with HTML, CSS, and JavaScript, 
deployed via GitHub Pages.

**Live site:** [brianinteractive.com](https://brianinteractive.com) ← replace with your actual domain

---

## About

This portfolio showcases 11+ years of browser-layer platform engineering, 
from real-time biometric data visualization at Sticky/Tobii to embedded 
JavaScript infrastructure and internal developer tooling at Wunderkind.

Built to be fast, accessible, and straightforward. No frameworks, no build 
pipeline overhead — just clean HTML, CSS, and vanilla JavaScript served 
directly from GitHub Pages.

---

## Running Locally

Clone the repo and open `index.html` directly in a browser. No server 
required.
```bash
git clone https://github.com/brianeng00/brianeng-portfolio.git
cd brianeng-portfolio
open index.html
```
